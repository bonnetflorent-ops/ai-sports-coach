# Analyse Concurrentielle — AI Sports Coach

> Marché du coaching sportif digital en 2025-2026

## Paysage concurrentiel

```
Haut de gamme (100-300€/mois)
│  Coachs humains
│  TrainingPeaks Premium + coach
│
├── Notre cible (15-20€/mois)
│   │  ★ AI Sports Coach (chat IA conversationnel)
│   │
│   ├── Runna (15,99€/mois)
│   ├── JOIN (14,99€/mois)
│   ├── Freeletics (9,99-16,99€/mois)
│   │
│   └── FizzUp (9,99€/mois)
│
Bas de gamme (gratuit-5€/mois)
   Apps génériques, PDF, YouTube
```

## Concurrent direct n°1 : Freeletics

| Aspect | Détail |
|--------|--------|
| **Prix** | 9,99-16,99€/mois |
| **Sports** | Fitness, poids de corps |
| **Chat IA** | ❌ Aucun |
| **Points forts** | Contenu vidéo de qualité, communauté, AI coach qui ajuste les répétitions (mais pas conversationnel) |
| **Points faibles** | Pas de sports d'endurance (course, vélo), personnalisation limitée au fitness, pas de dialogue |
| **Technique** | Adapte la difficulté des séances selon feedback, mais pas de conversation |
| **Marketing** | Marque forte, beaucoup de contenu, ambassadeurs |

**Notre avantage** : Freeletics ne parle pas. L'utilisateur ne peut pas dire "j'ai mal dormi" et avoir une adaptation intelligente. On attaque sur le dialogue + les sports d'endurance.

## Concurrent direct n°2 : Runna

| Aspect | Détail |
|--------|--------|
| **Prix** | 15,99€/mois |
| **Sports** | Running seulement (route, trail) |
| **Chat IA** | ❌ Pas de chat |
| **Points forts** | Excellente réputation (4.8★), plans structurés de qualité, intégration montres (Garmin, Apple, Coros), analyse de course |
| **Points faibles** | Running uniquement, plans fixes une fois générés, pas de dialogue, pas de multi-sport |
| **Technique** | Génère un plan course/piste, ajuste selon les sorties, mais ne dialogue pas |
| **Marketing** | "Runna vs coach humain", chiffres d'amélioration concrets |

**Notre avantage** : Runna est l'excellence en running mais zéro conversation. Un runner qui fait aussi du vélo n'a rien. Et si après 3 semaines il veut modifier son plan, il ne peut pas le dire — il doit tout re-générer. Nous : "change ma séance de demain, je suis fatigué" → fait.

## Concurrent direct n°3 : JOIN

| Aspect | Détail |
|--------|--------|
| **Prix** | 14,99€/mois |
| **Sports** | Cyclisme, running |
| **Chat IA** | ❌ Pas de chat |
| **Points forts** | Belle UI, data-driven, importe les données Strava/Garmin, multi-sport, plans adaptatifs |
| **Points faibles** | Coaching superficiel, adapte mais n'explique pas, pas de dialogue, personnalisation limitée |
| **Technique** | Analyse les données d'entraînement, ajuste les prochaines séances |
| **Marketing** | Axé data, intégration capteurs |

**Notre avantage** : JOIN adapte des séances mais ne t'explique pas POURQUOI. Pas de pédagogie. Un athlète veut comprendre. Nous on explique chaque choix, on dialogue.

## Concurrent direct n°4 : FizzUp

| Aspect | Détail |
|--------|--------|
| **Prix** | 9,99€/mois |
| **Sports** | Fitness |
| **Chat IA** | ❌ |
| **Points forts** | Marque française, contenu vidéo français, programmes variés |
| **Points faibles** | Coaching vidéo pré-enregistré, zéro personnalisation conversationnelle, pas d'endurance, IA quasi inexistante |

**Notre avantage** : FizzUp c'est des vidéos YouTube premium. Nous c'est un coach dans ta poche.

## Concurrent n°5 : TrainingPeaks (indirect)

| Aspect | Détail |
|--------|--------|
| **Prix** | Gratuit (basique) à 20€/mois (premium), + coach humain (100-300€) |
| **Sports** | Endurance (triathlon, cyclisme, running) |
| **Chat IA** | ❌ |
| **Points forts** | Référence pro, analytics avancés, WKO5, planification pro |
| **Points faibles** | Complexe, orientation pro/élite, sans coach humain c'est juste un outil d'analyse |
| **Technique** | Plateforme d'analyse de données d'entraînement |
| **Marketing** | Marque historique, communauté pro |

**Notre avantage** : TrainingPeaks sans coach humain c'est une coquille vide pour un amateur. Nous on remplace le coach humain.

## Menaces émergentes

- **Strava** : a les données de tout le monde. S'ils sortent un coach IA, c'est un risque majeur.
- **Garmin** : Garmin Coach existe mais basique. S'ils l'améliorent avec IA...
- **Apple Fitness+** : mise fort sur l'IA, intégration Watch, mais pour l'instant c'est fitness uniquement.
- **Startups IA coaching** : probablement des YC ou autres qui arrivent.

## Stratégie d'attaque

### Phase 1 : Lancement France
- Communautés Strava françaises
- Groupes Facebook cyclisme (200k+ membres cumulés)
- Reddit r/running, r/pedale
- Forums vélo (Velo101, OnVeloo)

### Phase 2 : Ambassadeurs
- 5-10 athlètes amateurs influents (pas des pros, des gens normaux)
- Programme "early adopter" avec prix réduit à vie

### Phase 3 : Contenu
- Blog articles scientifiques traduits en conseils pratiques
- Comparaisons honnêtes vs concurrents
- Témoignages early adopters

### Marketing à imiter
- **Style Runna** : chiffres d'amélioration concrets, benchmark vs coach humain
- **Style JOIN** : démo produit, intégrations, data viz
- **Différenciant** : "Ton coach te parle vraiment."

## Matrice SWOT

| Forces | Faiblesses |
|--------|------------|
| Chat IA conversationnel unique | Bootstrappé solo |
| Multi-sport | Pas de contenu vidéo |
| Qualité scientifique | Marque inconnue |
| Français | Pas de data d'utilisateurs |
| Intégrations (calendrier) | Budget marketing limité |

| Opportunités | Menaces |
|-------------|---------|
| Aucun concurrent sur le chat | Strava Coach IA |
| Marché coaching en croissance | Garmin Coach IA amélioré |
| IA de plus en plus accessible | Régulation EU AI Act |
| Français = marché moins concurrentiel | Copies rapides si succès |
